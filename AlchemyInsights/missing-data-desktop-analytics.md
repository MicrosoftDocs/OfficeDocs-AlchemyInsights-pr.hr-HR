# <a name="cant-create-a-compatibility-assessment"></a>Nije moguće stvoriti procjenu kompatibilnosti

Microsoft Endpoint upravitelj konfiguracije uspješno je konfiguriran uređaj, ali Desktop Analytics ne može stvoriti procjenu kompatibilnosti. Nema kompletan skup podataka za konfiguraciju uređaja (popis) ili instalirane aplikacije (inventar).

Taj se problem često automatski rješava kada se uređaj ponovno pokuša. Ako se nastavi, provjerite može li uređaj komunicirati s uslugom. Za više informacija pogledajte [krajnje točke](https://docs.microsoft.com/configmgr/desktop-analytics/enable-data-sharing#endpoints) i [podatke koji nedostaju](https://docs.microsoft.com/configmgr/desktop-analytics/monitor-connection-health#missing-data).
