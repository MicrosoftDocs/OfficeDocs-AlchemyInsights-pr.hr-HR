# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Prilagodba dozvola web-mjesta sustava SharePoint putem grupa sustava SharePoint 

Grupa sustava SharePoint zbirka je korisnika koji imaju isti skup dozvola za web-mjesta i sadržaj. Umjesto dodjele dozvola jednu po jednu osobu, možete koristiti grupe da biste jednostavno dodijelili istu razinu dozvola većem broja osoba odjednom.

[Prilagodba dozvola web-mjesta sustava SharePoint](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
