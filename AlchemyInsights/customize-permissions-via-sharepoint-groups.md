# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Prilagodba dozvola za Web-mjesto sustava SharePoint putem grupa sustava SharePoint 

SharePoint grupa zbirka je korisnika koji imaju isti skup dozvola za web-mjesta i sadržaj. Umjesto dodjeljivanja dozvola jednoj po osobi, pomoću grupa možete jednostavno dodijeliti istu razinu dozvole mnogim osobama odjednom.

[Prilagodba dozvola za web-mjesto sustava SharePoint](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
