# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Prilagođavanje dozvola web-mjesta sustava SharePoint putem grupa sustava SharePoint 

Grupa sustava SharePoint zbirka je korisnika kojima svi imaju isti skup dozvola za web-mjesta i sadržaj. Umjesto da dodijelite dozvole jednoj osobi istodobno, možete koristiti grupe za povoljno dodjeljivanje iste razine dozvola mnogima istodobno.

[Prilagodba dozvola web-mjesta sustava SharePoint](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
