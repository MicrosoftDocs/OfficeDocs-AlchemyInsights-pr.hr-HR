# <a name="gatewaynotreachable-when-setting-credentials"></a>Gatewaynotreu prilikom postavljanja vjerodajnica

Prilikom pokušaja postavljanja vjerodajnica za izvor podataka unutar servisa Power BI možete naići na Gatewaynotreu. Ovo bi mogao biti rezultat zastarjelog pristupnika. Instalirajte najnoviji Gateway i pokušajte ponovno.

Za više informacija:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
