# <a name="pdf-reader-app-in-microsoft-edge"></a>Aplikacija PDF Reader u pregledniku Microsoft Edge

Popularnost PDF datoteka ističe potrebu za pouzdanim, sigurnim i snažnim čitačem PDF-a koje Enterprise tvrtke može usvojiti. Čitač značajki obogaćenog PDF-a ugrađenog u Microsoft Edge (verzije 77 i novije verzije) upravo je takva aplikacija. Funkcionira u sustavima Windows i macOS i može se koristiti za otvaranje i označavanje PDF datoteka lokalnih, mrežnih ili ugrađenih na web-stranicama.

Dodatne informacije potražite u članku [čitač PDF-ova u programu Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-pdf).
