# <a name="profile-property-synchronization"></a>Sinkronizacija svojstava profila

Nije moguće mapirati dodatna svojstva za uvoz oglasa za sinkronizaciju iz servisa Azure Active Directory u aplikaciju korisničkog profila. Uvoz oglasa ograničen je na unaprijed konfiguriran skup svojstava radi garancije dosljedne performanse zadatka mjerača vremena.
  
[Saznajte više o svojstvima profila koja se sinkronizira](https://go.microsoft.com/fwlink/?linkid=875671)
