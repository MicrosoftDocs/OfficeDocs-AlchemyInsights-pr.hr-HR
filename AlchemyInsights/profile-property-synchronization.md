# <a name="profile-property-synchronization"></a>Sinkronizacija svojstava profila

Nije moguće mapirati dodatna svojstva za AD Import za sinkronizaciju iz servisa Azure Active Directory u aplikaciju korisničkog profila. Uvoz oglasa ograničen je na unaprijed konfigurirani skup svojstava kako bi se zajamčila dosljedna izvedba zadatka mjerača vremena.
  
[Saznajte više o svojstvima profila koja sinkroniziraju](https://go.microsoft.com/fwlink/?linkid=875671)
