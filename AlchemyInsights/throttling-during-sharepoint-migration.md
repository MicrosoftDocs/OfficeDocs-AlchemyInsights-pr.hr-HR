# <a name="sharepoint-migration-throttling"></a>Ograničavanje migracije sustava SharePoint

SharePoint Online koristi ograničavanje za održavanje optimalnih performansi i pouzdanosti servisa SharePoint Online. Ograničavanjem se ograničava broj korisničkih radnji ili istodobnih poziva (skriptom ili kodom) radi sprječavanja prekomjernog korištenja resursa. Dodatne informacije potražite u:

- [Tijekom migracije doživljavam loše performanse ili ograničavanje](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed#faq-and-troubleshooting)
