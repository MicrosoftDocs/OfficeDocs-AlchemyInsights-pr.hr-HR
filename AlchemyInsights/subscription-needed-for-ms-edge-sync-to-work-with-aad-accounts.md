# <a name="subscription-needed-for-microsoft-edge-sync-to-work-with-azure-active-directory-accounts"></a>Pretplata potrebna za Microsoft Edge Sync za rad s računima za Azure Active Directory

Sinkronizacija će funkcionirati s bilo kojom od sljedećih pretplata:

1. Azure Active Directory Premium (P1 i P2)
1. Microsoft 365 Business Premium
1. Office 365 E3 i iznad
1. Zaštita podataka za Azure (P1 i P2)
1. Sve pretplate na Office 365 Education:
    - Microsoftove aplikacije za studente ili fakultet
    - Exchange Online za studente ili fakultet
    - Office 365 a1 ili noviji
    - Microsoft 365 a1 ili noviji
    - Zaštita informacija o Azure P1 ili P2, za studente ili fakultet
