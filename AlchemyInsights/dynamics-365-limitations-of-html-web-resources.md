# <a name="webpage-html-web-resources-limitations"></a>Ograničenja resursa web web-stranica (HTML)

* HTML web-resursa ne može sadržavati kod koji se mora izvršiti na poslužitelju. ASP.NET stranica ne može prenijeti kao HTML web-resurse.

* HTML web-resurse možete prihvatiti samo ograničeni broj parametrima niza upita. Dodatne informacije: [Prosljeđivanje parametara HTML web-resursa](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
