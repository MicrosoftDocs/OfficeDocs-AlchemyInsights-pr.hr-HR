# <a name="windows-fresh-start"></a>Novi Start u sustavu Windows

Akcija uređaja sustava Windows Fresh Start nudi alternativu za brisanje i povlačenje uređaja na uređaju uz mogućnost zadržavanja stanja upisa uređaja zajedno s aplikacijama i postavkama korisničke kućne mape.

Dodatne informacije potražite u članku [korištenje novog početka za vraćanje izvornih postavki uređaja sa sustavom Windows 10 pomoću aplikacije Intune](https://docs.microsoft.com/intune/device-fresh-start).
