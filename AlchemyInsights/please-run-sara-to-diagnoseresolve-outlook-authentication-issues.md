# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Koristite SaRA za dijagnosticiranje i rješavanje problema Outlook provjere autentičnosti

Vrlo je preporučljivo koristite u [Outlook stalno za moju lozinku](https://aka.ms/SaRA-OutlookPwdPrompt) dijagnostički na utječe stroja poteškoća gdje se Outlook neprestano pita za lozinku. Ove [SaRA](https://diagnostics.office.com/#/) dijagnostički ne automatizirani provjerava i vraća mogućih rješenja za korištenje adresa otkriveni problemi.
