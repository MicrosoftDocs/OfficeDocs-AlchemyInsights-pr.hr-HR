# <a name="microsoft-search-setup-guide"></a>Vodič za postavljanje Microsoftova pretraživanja

Upotrijebite [Vodič za postavljanje Microsoftova pretraživanja](https://go.microsoft.com/fwlink/?linkid=2153798) da biste postavili i konfigurirali Microsoft Search. Želite li ga upravljati grupi korisnika ili je odjaviti svima u tvrtki ili ustanovi. Dodijeljućete administratore pretraživanja i uređivača pretraživanja, a zatim prilagoditi doživljaj pretraživanja korisnicima uz odgovore i dodatne mogućnosti, kao što je dodavanje proširenja servisa Bing u Chrome ili postavljanje aplikacije Bing kao zadane tražilice.
