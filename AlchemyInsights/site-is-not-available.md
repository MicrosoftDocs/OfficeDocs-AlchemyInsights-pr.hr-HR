# <a name="site-is-not-available"></a>Web-mjesto nije dostupno

Ako SharePoint ili OneDrive web-mjesta nije dostupan za više korisnika, možda postoji problem privremeni servisa. Provjera [stanja servisa nadzorne ploče](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) da biste vidjeli ako vaše klijentske utjecati.
