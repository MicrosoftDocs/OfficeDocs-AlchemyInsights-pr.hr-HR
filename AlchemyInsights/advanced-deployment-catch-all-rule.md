# <a name="deploy-microsoft-365-apps"></a>Implementacija aplikacija sustava Microsoft 365

Za implementaciju Microsoftova 365 aplikacija na više krajnjih korisnika putem alata za implementaciju sustava Office, upravitelja konfiguracije programa Microsoft Endpoint, kao dio diska za operacijski sustav ili na zajedničkom računalnom scenariju pomoću sustava Windows Server s udaljenom radnom površinom (RDS) ili servisa Terminal Services, razmotrite korištenje vodiča po koracima u programu **microsoft 365 Apps za Enterprise Deployment Advisor**.
  
Da biste pokrenuli početak rada, provjerite jeste li prijavljeni kao administrator korisnika ili globalni administrator, a zatim kliknite ovdje: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
