# <a name="deploy-microsoft-365-apps"></a>Implementacija aplikacija sustava Microsoft 365

Za uvođenje aplikacija sustava Microsoft 365 na više krajnjih korisnika putem alata za implementaciju sustava Office, Upravitelj konfiguracije krajnje točke Microsoft Endpoint, kao dio slike diska operacijskog sustava ili u scenariju zajedničkog računala pomoću sustava Windows Server s udaljenom radnom površinom services (RDS) ili servisima terminala razmislite o korištenju vodiča korak po korak u **Microsoft 365 Aplikacijama za savjetnik za implementaciju poduzeća**.
  
Da biste započeli, provjerite jeste li prijavljeni kao administrator klijenta ili globalni administrator, a zatim kliknite ovdje: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
