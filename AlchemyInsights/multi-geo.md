# <a name="microsoft-365-multi-geo"></a>Microsoft 365 Multi-Geo

Uz Microsoft 365 Multi-Geo vaša tvrtka ili ustanova može proširiti svoju prisutnost u sustavu Microsoft 365 na više geografskih regija i/ili zemalja unutar postojećeg klijenta. Dodatne informacije potražite na vezama u nastavku.

- [Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/office-365-multi-geo)
- [Mogućnosti više mogućnosti na servisima OneDrive i SharePoint Online](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-onedrive-and-sharepoint-online-in-office-365)
- [Mogućnosti za više geoprograma u sustavu Exchange Online](https://docs.microsoft.com/office365/enterprise/multi-geo-capabilities-in-exchange-online)
- [Planiranje za Microsoft 365 Multi-Geo](https://docs.microsoft.com/office365/enterprise/plan-for-multi-geo)
- [Konfiguriranje multi-geo](https://docs.microsoft.com/office365/enterprise/multi-geo-tenant-configuration)
