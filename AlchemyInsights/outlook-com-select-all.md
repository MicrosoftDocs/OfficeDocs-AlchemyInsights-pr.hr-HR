# <a name="how-to-select-all-in-outlookcom"></a>Kako "odabrati sve" u Outlook.com

Sve možete odabrati samo ako su sve poruke vidljive. Time se sprječavaju slučajne akcije na porukama koje ne možete vidjeti.

**CTRL + klik** odabire više pojedinačnih poruka.

**SHIFT + klik** odabire sve poruke između prve i posljednje poruke koju odaberete.
