# <a name="problems-setting-up-an-email-account-in-outlook"></a>Problemi s postavljanjem računa e-pošte u programu Outlook

Ako imate problema s postavljanjem računa sustava Exchange u programu Outlook, alat [za postavljanje računa programa Outlook](https://aka.ms/SaRA-OutlookSetupProfile) možete pokrenuti na zahvaćenom računalu. Obavljat će različite provjere i pružiti moguća rješenja za problem postavljanja računa.
  
Ako i dalje imate problema, pročitajte članak [Outlook ne može postaviti novi profil pomoću automatskog otkrivanja sustava Exchange za poštanski sandučić sustava Exchange Online](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover).
