# <a name="duplicate-object-synchronization-errors"></a>Pogreške prilikom sinkronizacije duplikata objekta

Ako tijekom sinkronizacije primite neku od sljedećih pogrešaka:

- Sinkronizirani objekt s istom proxy adresom već postoji u direktoriju Microsoftova internetskih servisa.

- Slična poruka za UserPrincipalName.

Pomoću nove dijagnostike servisa AADConnect riješite duplicirane pogreške u objektu. Dodatne informacije potražite u članku [dijagnosticiranje i korekcija dupliciranih pogrešaka pri sinkronizaciji atributa](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
