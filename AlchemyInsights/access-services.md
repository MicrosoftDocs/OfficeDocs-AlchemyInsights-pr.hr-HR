# <a name="access-services"></a>Access Services

Značajka Access Services u SharePoint Online je u tijeku mirovini. Novi pristup temelji web apps i web-baze podataka više ne mogu stvoriti i postojeći web apps i web-baze podataka će se isključiti po Travanj, 2018. Radne površine baze podataka programa Access (.accdb) neće utjecati Ova promjena. Preporučujemo da Microsoft PowerApps kao alternativu pristup web-apps i web-baze podataka. 
  
[Saznajte više o Microsoft PowerApps](https://powerapps.microsoft.com/)
