# <a name="change-the-font-size-in-a-message"></a>Promjena veličine fonta u poruci

1. Ako ste poruku već upisali, odaberite tekst.
2. Odaberite ikonu  **oblikovanja** ispod poruke (to je prva ikona).
3. Odaberite ikonu  **veličine fonta**  iznad poruke (to je Sedma ikona).
4. Odaberite veličinu.
