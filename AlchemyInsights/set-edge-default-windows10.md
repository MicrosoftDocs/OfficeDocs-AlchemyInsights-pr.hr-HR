# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Postavljanje preglednika Microsoft Edge kao zadanog preglednika na uređaju sa sustavom Windows 10

Evo i kako:

1. Odaberite **gumb Start,** a zatim unesite **Zadane aplikacije**.

1. U rezultatima pretraživanja odaberite **Zadane aplikacije**.

1. U **odjeljku Web-preglednik** **odaberite Microsoft Edge**.
