# <a name="why-cant-i-edit-this-file"></a>Zašto ne mogu uređivati tu datoteku?

Ako je dokument, radni list ili prezentacija zaključana radi uređivanja i ne može se uređivati u aplikacijama sustava Windows za stolna računala, možda datoteku uređuje netko drugi ili je odjavio.

Pogledajte neke od razloga zbog kojih možda nećete moći [uređivati datoteku](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
