# <a name="why-cant-i-edit-this-file"></a>Zašto ne mogu uređivati ovu datoteku?

Ako je dokument, radni list ili prezentacija zaključani za uređivanje i nije ga moguće uređivati u aplikacijama sustava Windows za stolna računala, možda je datoteka uređuje netko drugi ili je datoteka odjavljena.

Pogledajte neki od razloga [zbog kojih možda nećete moći uređivati datoteku](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
