# <a name="removing-the-last-on-premises-exchange-server"></a>Uklanjanje posljednjeg poslužitelja Exchange lokalno

Prije uklanjanja posljednji Exchange server iz vaše organizacije lokalno, provjerite je li razumijete implikacije time pomoći odlučite trebate taj zadnje poslužitelj za druge namjene. Dodatne informacije potražite odjaviti [kako i kada decommission lokalno Exchange poslužitelji u hibridno uvođenja](https://technet.microsoft.com/en-us/library/dn931280%28v=exchg.150%29.aspx).
