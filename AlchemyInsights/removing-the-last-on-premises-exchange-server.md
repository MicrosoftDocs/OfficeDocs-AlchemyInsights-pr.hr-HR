# <a name="removing-the-last-on-premises-exchange-server"></a>Uklanjanje zadnjeg lokalnog poslužitelja sustava Exchange

Prije uklanjanja posljednjeg poslužitelja sustava Exchange iz lokalne tvrtke ili ustanove, trebali biste biti sigurni da razumijete implikacije toga da biste lakše odlučili ako vam je potreban taj zadnji poslužitelj za bilo koju drugu svrhu. Dodatne informacije potražite [u članku kako i kada poništiti Komisiju lokalnih poslužitelja sustava Exchange u hibridnu implementaciju](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
