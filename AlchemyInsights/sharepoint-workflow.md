# <a name="troubleshoot-workflows-in-sharepoint"></a>Rješavanje problema s tijekovima rada u sustavu SharePoint

**Otklanjanje poteškoća s tijekovima rada i ispravljanje pogrešaka u sustavu SharePoint**

Pogledajte [Otklanjanje poteškoća s provjerama valjanosti tijeka rada sustava SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio) i ispravljanje [pogrešaka tijekova rada sustava SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).

**Power Automat**

Ako tek započinjete s tijekovima rada, razmislite o korištenju značajke [Power Automate](https://docs.microsoft.com/power-automate/modern-approvals) umjesto klasičnih rješenja tijeka rada sustava SharePoint.

Dodatne informacije potražite u odjeljku [Tijekovi rada sustava SharePoint koji se povlače](https://docs.microsoft.com/alchemyinsights/sharepoint-workflows-retiring).
