# <a name="identify-your-tenant-id"></a>Identificiranja svog ID-a stanara

Pronađite ID tvrtke ili ustanove tvrtke Microsoft 365 u centru za administratore Azure.

1. Na [stranici svojstva](https://aka.ms/AzurePropertiesPage)otvorite okvir **ID direktorija** .

2. Po potrebi kopirajte ID.

Dodatne informacije potražite u članku [Pronalaženje ID-a tvrtke ili ustanove tvrtke Microsoft 365](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id).
