# <a name="set-up-azure-active-directory"></a>Postavljanje servisa Azure Active Directory

[Vodič za postavljanje Azure ad](https://go.microsoft.com/fwlink/?linkid=2134390) nudi informacije koje osiguravaju da vaša tvrtka ili ustanova ima jaku sigurnosnu zakladu. U ovom ćete vodiču postaviti početne značajke, kao što su Azure (Azure RBAC) za administratore, Azure AD Connect za lokalni direktorij i Azure AD Connect zdravlje, tako da možete pratiti zdravlje hibrida identiteta tijekom automatskog sinkronizacije.

Sadrži i bitne informacije o omogućivanju ponovnog postavljanje lozinke za samoposluživanje, uvjetnom pristupu i integriranom prijavom drugih proizvođača, uključujući neobavezno Napredno zaštitu identiteta i automatizaciju korisničkih resursa.
