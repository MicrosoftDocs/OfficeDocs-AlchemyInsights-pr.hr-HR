# <a name="azure-active-directory-plans-comparison"></a>Usporedba tarife za Azure Active Directory

Azure Active Directory isporučuje se u četiri izdanja: besplatno, Office 365 Apps, Premium P1 i Premium P2.

Besplatno izdanje sadrži pretplatu na Azure. Premium izdanja dostupne su putem Microsoftova predstavnika, [otvorenog količinskog licenciranja](https://go.microsoft.com/fwlink/?linkid=2110873)i [programa davatelja rješenja za oblak](https://go.microsoft.com/fwlink/?LinkId=614968&clcid=0x409).

Azure i Office 365 pretplatnici mogu kupiti i Azure Active Directory Premium P1 i P2 online. Dodatne informacije potražite u članku [Azure Active Directory značajke i cijene](https://go.microsoft.com/fwlink/?linkid=2081447).
