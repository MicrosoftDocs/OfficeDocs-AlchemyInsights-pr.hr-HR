# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Na platformi Windows Server u programu Microsoft Defender za krajnju točku

Za operacijski sustav Windows Server Microsoft Defender za krajnju točku nudi napredne mogućnosti za otkrivanje i istraživanje napada. Te su mogućnosti neprimjetno dostupne putem konzole Microsoft Defender Security Center.

Defender za krajnju točku podržava sljedeće verzije sustava Windows Server:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (SAC) verzija 1803 i noviji
- Windows Server 2019 i noviji
- Osnovno izdanje sustava Windows Server 2019

Dodatne informacije potražite u članku [poslužitelji sustava Windows u sustavu Microsoft Defender za servis krajnje točke](https://go.microsoft.com/fwlink/?linkid=2143627).
