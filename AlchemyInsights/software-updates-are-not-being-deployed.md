# <a name="software-updates-are-not-being-deployed"></a>Ažuriranja softvera nisu uvedena

Ako ste softverska ažuriranja konfigurirali putem prstena za ažuriranje sustava Windows 10, ali se ažuriranja ne uvode, pokušajte sljedeće:  

- Razmislite o promjeni servisiranja sustava Windows iz **polugodišnje** vrste izdanja kanala u strožu, češćiju vrstu izdanja.
- Provjerite razdoblje odgode ažuriranja **kvalitete** i **ažuriranja značajki**. Razdoblje odgode može dovesti do kašnjenja u ažuriranjima do 180 dana.
