# <a name="sharepoint-online-calendar-overlay"></a>Preklapanje kalendara sustava SharePoint Online

Drugi kalendar sustava SharePoint možete prekriti sve dok su oba kalendara sustava SharePoint u istoj zbirci web-mjesta. Mogućnost korištenja prekrivanja kalendara u kalendaru sustava SharePoint Online ili SharePoint Server koji je dohvaćen iz sustava Exchange Online nije podržana.

SharePoint Online nudi moderan doživljaj s kalendarima grupa sustava Microsoft 365. Dodatne informacije [potražite u odjeljku Korištenje web-dijela Grupni kalendar](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce).
