# <a name="sharepoint-online-calendar-overlay"></a>Preklapanje kalendara sustava SharePoint Online

Možete prekriti drugi kalendar sustava SharePoint sve dok se kalendari kalendara sustava SharePoint nalaze u istoj zbirci web-mjesta. Mogućnost korištenja preklapanja kalendara na servisu SharePoint Online ili kalendaru sustava SharePoint Server koja se dohvaća iz sustava Exchange Online nije podržana.

SharePoint Online nudi moderno iskustvo u grupnim kalendarima programa Microsoft 365. Dodatne informacije potražite u članku [korištenje web-dijela kalendar u grupi](https://support.microsoft.com/en-us/office/use-the-group-calendar-web-part-eaf3c04d-5699-48cb-8b5e-3caa887d51ce).
