# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Dodavanje proširenja u Microsoft Edge iz Chrome web-trgovine

Da biste ekstenziju dodali u Microsoft Edge iz Chrome web-trgovine, dovršite sljedeće korake:

1. Idite u Chrome web-trgovinu.
2. U Banneru pri vrhu stranice odaberite **Dopusti ekstenzije iz drugih trgovina**.
3. Odaberite ekstenziju koju želite dodati, a zatim kliknite **Dodaj u Chrome**.
4. U dijaloškom okviru za potvrdu pažljivo pregledajte izmjene koje proširenje može izvršiti, a zatim kliknite **Dodaj proširenje**.
Konačni dijaloški okvir potvrdit će da je ekstenzija dodana.
