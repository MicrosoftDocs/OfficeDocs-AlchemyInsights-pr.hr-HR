# <a name="changes-to-external-user-access"></a>Promjene vanjskog korisničkog pristupa

Počevši od ožujka 23, 2018 vanjski korisnici više neće moći pristupiti sadržaju koji je zajednički podijeljen sa svima, svim provjerenim korisnicima ili svim korisnicima obrazaca, čak i ako su pozvani ili dodani u ovlaštene grupe. Vanjski korisnici moći će pristupiti samo sadržaju koji je podijeljen s korisnikom ili s grupama kojima korisnik pripada. Ako želite zadržati postojeće postavke, pogledajte:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
