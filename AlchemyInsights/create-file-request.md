# <a name="how-to-create-a-file-request"></a>Kako stvoriti zahtjev za datotekom

Značajka zahtjeva za datotekom u servisu OneDrive omogućuje vam odabir mape u kojoj drugi mogu prenositi datoteke pomoću veze koju ste im poslali. Korisnici koje zatražite od datoteka ne mogu vidjeti što je u mapi--oni mogu prenositi samo datoteke na njega.

[Stvaranje zahtjeva za datotekom](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af)
