# <a name="distribution-group-not-showing-in-groups-list"></a>Grupa raspodjele ne prikazuje na popisu grupe

Može potrajati do 60 minuta prije nego se pojavi novi popis raspodjele u centru za Exchange admin centar.
  
Ako želite dodati članove ili odmah napravite druge uređivanja u [Exchange Admin centar](https://outlook.office365.com/ecp/?rfr=Admin_o365&amp;exsvurl=1&amp;mkt=en-US.aspx)koristite značajku grupe.
