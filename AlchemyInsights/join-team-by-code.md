# <a name="join-a-team-by-code"></a>Pridruživanje timu po kodu

Microsoft je svjestan problema kada korisnici ne mogu stvoriti **tim** kod ili se pridružiti timu pomoću značajke **pridruživanja prema kodu** . Kao zaobilazno rješenje, vlasnici tima mogu ručno dodavati korisnike po potrebi.
