# <a name="fix-windows-update-errors"></a>Otklanjanje pogrešaka u servisu Windows Update

Ako se prilikom preuzimanja i instaliranja servisa Windows Update pojavi kod pogreške, **alat za otklanjanje poteškoća s ažuriranjem** može pomoći u rješavanju problema.

- [Preuzimanje alata za otklanjanje poteškoća s ažuriranjem za Windows 10](https://support.microsoft.com/help/4027322/windows-update-troubleshooter)
- Pokrenite alat za otklanjanje poteškoća. Kada je to gotovo, najbolje je vježbati ponovno pokrenuti uređaj. Zatim provjerite ima li novih ažuriranja tako da [počnete > postavke > update & Security > Windows Update](ms-settings:windowsupdate).

Ako i dalje ne funkcioniraju, provjerite [Windows Update: najčešća pitanja](https://support.microsoft.com/help/12373/windows-update-faq) o dodatnim informacijama.
