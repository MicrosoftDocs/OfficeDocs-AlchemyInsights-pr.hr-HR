# <a name="add-a-sharepoint-calendar"></a>Dodavanje kalendara sustava SharePoint

Kalendari sustava SharePoint klasične su vrste popisa koje se ponekad miješaju u grupne kalendare programa Microsoft 365.
 
Da biste stvorili kalendar sustava SharePoint, slijedite ove korake:
 
1.  Odaberite ikonu zupčanika **Postavke** , a zatim odaberite **Dodavanje aplikacije**.
2.  U okvir Traženje **aplikacije** potražite unos **kalendara** pa pritisnite ENTER.
3.  Odaberite aplikaciju **Kalendar**koji se naziva, a zatim unesite naziv kalendara.

Dodatne informacije o grupama sustava Microsoft 365 potražite [u članku upoznavanje s grupama microsoft 365](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
