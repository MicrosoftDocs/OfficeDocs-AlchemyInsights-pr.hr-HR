# <a name="create-a-site"></a>Stvaranje web-mjesta

- Idite na [stranici web-mjesta u centru za administraciju Microsoft 365](https://portal.office.com/adminportal/home#/SitesList)i odaberite **Dodaj web-mjesta**. 
    
- Novi centar admin SharePoint, na stranici web-mjesta, kliknite **Stvori**. 
    
 **Napomena:** Nije moguće stvoriti novo web-mjesto koje ima isti URL s postojećeg web-mjesta. 
  
Dodatne informacije o stvaranju web-mjesta u odjeljku [Stvaranje web-mjesta](https://go.microsoft.com/fwlink/?linkid=866295).
  
Info o dopuštanju korisnicima stvaranje vlastitih web-mjesta, pogledajte [Upravljanje stvaranja web-mjesta u SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
