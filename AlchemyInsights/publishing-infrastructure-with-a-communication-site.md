# <a name="publishing-infrastructure-with-a-communication-site"></a>Izdavačka infrastruktura s komunikacijskim web-mjestom


Omogućivanje značajke infrastrukture za objavljivanje sustava SharePoint Server **nije podržano** za komunikacijska web-mjesta. 
  
Dodatne informacije potražite u [odjeljku Prelazak s web-mjesta za objavljivanje na web-mjesta za komunikaciju](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
