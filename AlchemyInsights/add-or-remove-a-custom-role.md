# <a name="add-or-remove-a-custom-role"></a>Dodavanje i uklanjanje prilagođene uloge

Upute za upravljanje resursima Azure potražite u članku [Dodavanje i uklanjanje dodjela uloga Azure pomoću portala Azure](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Da biste upravljali prilagođenim ulogama za Azure Active Directory (Azure AD), pročitajte članak [prilagođene administratorske uloge u servisu Azure Active Directory (pretpregled)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
