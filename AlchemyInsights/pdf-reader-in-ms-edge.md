# <a name="pdf-reader-in-microsoft-edge"></a>Čitač PDF-ova u programu Microsoft Edge

Popularnost PDF datoteka ističe potrebu za pouzdanim, sigurnim i snažnim čitačem PDF-a koje Enterprise tvrtke može usvojiti. Čitač značajki obogaćenog PDF-a ugrađenog u Microsoft Edge (verzije 77 i novije) upravo je takva aplikacija. Funkcionira u sustavima Windows i macOS i može se koristiti za otvaranje i označavanje PDF datoteka lokalnih, mrežnih ili ugrađenih na web-stranicama.

Dodatne informacije potražite u članku [čitač PDF-ova u programu Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2140005).
