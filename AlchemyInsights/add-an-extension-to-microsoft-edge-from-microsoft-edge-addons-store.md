# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Dodavanje proširenja u Microsoft Edge iz trgovine dodatka Microsoft Edge

Da biste u Microsoft Edge dodali produžetak iz trgovine dodatka Microsoft Edge, učinite sljedeće:

- Otvorite Microsoft Edge.
- Odaberite **... > ekstenzije > [dohvaćanje proširenja za Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2136408)**.
- Odaberite proširenje koje želite dodati, a zatim odaberite **Dohvati**.
- U dijaloškom okviru za potvrdu pažljivo pregledajte promjene koje može izvršiti proširenje, a zatim odaberite **Dodaj ekstenziju**.

Nakon toga ćete dobiti potvrdu da je dodatak dodan.
