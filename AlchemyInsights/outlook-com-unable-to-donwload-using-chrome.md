# <a name="cant-download-files-in-chrome"></a>Nije moguće preuzeti datoteke u Chromeu

**Broj izdanja** Svjesni smo problema koji sprječava neke korisnike da mogu preuzeti datoteke prilikom upotrebe preglednika Google Chrome. 

**Zaobilazno rješenje** Koristite [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) ili neki drugi web-preglednik.
Saznajte više o preuzimanju [datoteka i Outlook.com privitaka](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
