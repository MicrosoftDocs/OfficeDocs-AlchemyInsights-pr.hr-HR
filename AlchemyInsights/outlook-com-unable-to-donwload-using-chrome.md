# <a name="cant-download-files-in-chrome"></a>Nije moguće preuzeti datoteke u pregledniku Chrome

**Problem** Svjesni smo problema koji korisnicima onemogućuje mogućnost preuzimanja datoteka prilikom korištenja preglednika Google Chrome. 

**Zaobilazno rješenje** Koristite [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) ili neki drugi web-preglednik.
Saznajte više o [preuzimanju datoteka i privitaka za Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
