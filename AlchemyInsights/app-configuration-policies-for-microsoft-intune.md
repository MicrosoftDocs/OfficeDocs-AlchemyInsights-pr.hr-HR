# <a name="app-configuration-policies-for-microsoft-intune"></a>Pravila konfiguracije aplikacije za Microsoft Intune

Pravila konfiguracije aplikacija u programu Microsoft Intune pružaju konfiguracijske postavke za aplikacije sustava iOS ili Android. Te konfiguracijske postavke omogućuju prilagodbu aplikacije. Podrška za pravila konfiguracije ograničena je verzijama i aplikacijama operacijskog sustava.

Dodatne informacije potražite u [odjeljku Pravila konfiguracije aplikacija za Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
