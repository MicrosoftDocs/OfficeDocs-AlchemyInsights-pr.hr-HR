# <a name="enable-versioning-for-a-list-or-library"></a>Omogućavanje određivanja verzije za popis ili biblioteku

1. Idite na popis ili biblioteku.
    
2. Kliknite **Postavke**i zatim kliknite **Postavke biblioteke** ili **popisa postavke**.
    
3. Kliknite **Postavke određivanja verzija**.
    
4. Odaberite postavke koje želite, a zatim kliknite **u redu**.
