# <a name="sharepoint-hub-site-navigation"></a>Navigacije web-mjesta SharePoint koncentrator

Na vrhu web-mjesto koncentratora i sve pridružene web-mjesta, iznad navigacije web-mjesta je navigacijske trake web-mjesto koncentratora. Ako ste vlasnik web-mjesta koncentrator, možete prilagoditi [navigacijske trake](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav). 

**Napomena:** Preporuča se odabir komunikacije web-mjesta ili web-mjesto tima pomoću Moderna predloška kao web-mjesto koncentratora. Ako koristite klasični timskog web-mjesta koncentrator navigacije će se pojaviti samo na stranicama Moderna i koncentrator postavke web-mjesta će se pojaviti samo na stranici Moderna.
