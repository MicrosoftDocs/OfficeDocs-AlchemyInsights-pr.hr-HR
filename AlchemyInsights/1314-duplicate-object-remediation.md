# <a name="duplicate-object-synchronization-errors"></a>Pogreške u sinkronizaciji duplikata objekt

Ako primite jednu od sljedećih pogrešaka tijekom sinkronizacije:
  
- Sinkronizirane objekt s iste adrese proxy već postoji u direktoriju Microsoft Online Services.
    
- Slična poruka UserPrincipalName.
    
Koristite novi dijagnostički stanja AADConnect za rješavanje duplikata objekt pogrešaka. Za dodatne informacije pogledajte [Dijagnosticiranje i remediate pogreške pri sinkronizaciji dupliciranu atribut](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-health-diagnose-sync-errors).
