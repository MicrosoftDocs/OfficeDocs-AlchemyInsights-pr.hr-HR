# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Javna mapa uklonjena Outlook webu (OWA) favorita

Poznati je problem s uklonjenim javnim mapama iz favorita u aplikaciji OWA pri svakoj odjavi ili osvježavanju zaslona. Problem će uskoro biti riješen. Do tada ponovno dodajte mapu za pristup iz aplikacije OWA.
