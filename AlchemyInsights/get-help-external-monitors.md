# <a name="get-help-with-external-monitors"></a>Pomoć za vanjske monitore

Postavke projekta možete jednostavno otvoriti pritiskom na sljedeće tipke u isto vrijeme:

- Tipka sa sustavom Windows (gumb Start logotipa sustava Windows) + P
- Odaberite **duplikat**, **proširivanje** ili **drugi zaslon samo** ovisno o vašim preferencama.

Ako imate problema s vanjskim monitorima, isprobajte ove korake za otklanjanje poteškoća:

- Isključite kabel koji povezuje PC s monitorom i ponovno ga priključite.
- Isključite sve uređaje koji nisu neophodni na PC-ju (primjerice adaptere ili dokove).

Pročitajte više o Microsoftovoj podršci:

- [Postavljanje dvostrukih monitora u sustavu Windows 10](https://support.microsoft.com/windows/set-up-dual-monitors-on-windows-10-3d5c15dc-cc63-d850-aeb6-b41778147554)
- [Korištenje više monitora u sustavu Windows 10](https://support.microsoft.com/windows/how-to-use-multiple-monitors-in-windows-10-329c6962-5a4d-b481-7baa-bec9671f728a)
