# <a name="upload-process-overview-in-microsoft-stream"></a>Pregled postupka prijenosa u programu Microsoft Stream

Kad god prenosite videozapis, Microsoft Stream automatski ga kodira u više bittarifa da bi se videozapis mogao pokrenuti pomoću prilagodljivog strujanja bitrate. Microsoft Stream optimizira bitrate i kvalitetu reproduciranja videozapisa na temelju dostupne propusnost preglednika.

- [Pregled postupka prijenosa u programu Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
