# <a name="how-to-buy-a-copy-of-windows-10"></a>Kupnja kopije Windows 10

Ažurirane cijene i upute za kupnju i preuzimanje pune verzije Windows 10, idite na [Microsoft Store](https://www.microsoft.com/store/b/windows).

**Napomena:** Besplatna Windows 10 nadogradnja putem aplikacije Nabavite Windows 10 (GWX) završila je 29. srpnja 2016.

**Važno:** Ako koristite Windows 7, Microsoft preporučuje da se pomaknete na novi PC sa sustavom Windows 10 jer je Windows 7 završio životni ciklus podrške 14. siječnja 2020.
