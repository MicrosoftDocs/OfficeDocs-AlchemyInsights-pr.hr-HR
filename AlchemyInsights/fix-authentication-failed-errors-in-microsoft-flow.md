# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Ispravljanje pogrešaka "Provjera autentičnosti nije uspjela" u programu Microsoft Flow

Ako primite poruku o pogrešci "Provjera autentičnosti nije uspjela" kada se pokušate prijaviti u Microsoft Flow, to je zato što je servis Flow onemogućen u servisu Azure Active Directory. Pojedinosti i korake za ispravljanje pogreške potražite [u odjeljku Ne možete se prijaviti u Microsoft Flow i primiti pogrešku "Provjera autentičnosti nije uspjela".](https://support.microsoft.com/help/4316891)
