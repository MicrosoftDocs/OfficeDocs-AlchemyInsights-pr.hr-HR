# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Pronalaženje rezultata u pretraživanju servisa Bing pomoću aplikacija za Azure Logic

Pomoću tražilice Bing unutar logičke aplikacije možete automatizirati obradu rezultata pretraživanja i učiniti te rezultate dostupnima za druge akcije. Možete, primjerice, pronaći stavke vijesti, a zatim ih cvrkut objaviti kao tweets u sažetku sadržaja.

Dodatne informacije potražite u članku [Pronalaženje rezultata u pretraživanju servisa Bing pomoću aplikacija za logiku Azure](https://go.microsoft.com/fwlink/?linkid=2151928).
