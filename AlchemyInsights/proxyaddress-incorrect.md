# <a name="proxyaddress-incorrect"></a>ProxyAddress nije ispravan

Kada se objekt sinkronizira sazure AD, vrijednosti koje su navedene u atributu proxyAddresses u servisu Active Directory uspoređuju se s pravilima Azure AD-a, a zatim se atribut proxyAddresses popunjava u Azure AD. Stoga vrijednosti atributa proxyAddresses za objekt u servisu Active Directory možda nisu iste kao vrijednosti atributa proxyAddresses u Azure AD.
  
Dodatne informacije o popunjavanju proxyadrese potražite [u odjeljku Kako se atribut proxyAddress popunjava u Azure AD.](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad)
