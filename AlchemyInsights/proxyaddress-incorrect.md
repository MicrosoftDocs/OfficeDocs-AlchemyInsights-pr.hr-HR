# <a name="proxyaddress-incorrect"></a>Neispravan ProxyAddress

Kada objekt se sinkronizira Azure AD, vrijednosti koje su navedene u proxyAddresses atribut u Active Directory uspoređuju s Azure AD pravila i zatim popunjavaju atributa proxyAddresses u Azure AD. Stoga vrijednosti atributa proxyAddresses za objekt u Active Directory možda nije jednak vrijednosti atributa proxyAddresses u Azure AD.
  
Da biste saznali više o kako popunjavaju u proxyaddress pogledajte [kako se popunjava proxyAddress atribut u Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
