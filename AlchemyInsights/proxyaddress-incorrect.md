# <a name="proxyaddress-incorrect"></a>ProxyAddress nije točan

Kada se objekt sinkronizira s Azure AD, vrijednosti koje su navedene u atributu Proxyadrese u servisu Active Directory uspoređuju se s pravilima servisa Azure AD, a zatim se atribut Proxyadresa popunjava u servisu Azure AD. Stoga vrijednosti atributa Proxyadrese za objekt u servisu Active Directory možda neće biti iste kao vrijednosti atributa Proxyadrese u servisu Azure AD.
  
Da biste saznali više o tome kako je proxyAddress popunjen, pročitajte članak [kako se atribut proxyAddress popunjava u servisu Azure ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
