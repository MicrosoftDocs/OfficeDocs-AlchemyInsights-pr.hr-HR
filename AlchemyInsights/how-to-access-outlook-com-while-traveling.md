# <a name="how-to-access-outlookcom-while-traveling"></a>Kako pristupiti Outlook.com tijekom putovanja

Za pristup Outlook.com tijekom putovanja, provjerite je li vaš Microsoft račun sigurnosti info - zamjenski e-pošte adresu ili telefonski broj gdje možete dobiti sigurnosni šifre - ažurna. Da biste saznali više, pogledajte [kako pristupiti Outlook.com kada putujete](https://support.office.com/article/c44f16da-7156-4890-853c-286aafeda87e?wt.mc_id=Office_Outlook_com_Alchemy).
