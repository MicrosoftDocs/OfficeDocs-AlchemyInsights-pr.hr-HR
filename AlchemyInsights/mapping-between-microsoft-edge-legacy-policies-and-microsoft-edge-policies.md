# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Saznajte više o preslikavanju između Microsoftovih naslijeđenih pravilnika i Microsoft Edge pravilnika

Pravila grupe za Microsoft Edge Legacy (verzije 45 i starije) i mape postavke upravljanja mobilnim uređajima (MDM) na relevantne pravila sustava Microsoft Edge, verzija 80. Dodatne informacije potražite u članku [mapiranje pravilnika u pregledniku Microsoft Edge u pregledniku Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).
