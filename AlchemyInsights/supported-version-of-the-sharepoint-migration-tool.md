# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Podržana verzija alata za migraciju sustava SharePoint



Alat za migracije sustava SharePoint trenutno podržava samo migracije iz sustava SharePoint 2013. Verzije 2010 i 2016 nisu podržane u ovom trenutku.
  
Dodatne informacije potražite [u okviru uvođenja alata za migraciju sustava SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
