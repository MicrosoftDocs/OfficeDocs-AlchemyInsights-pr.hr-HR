# <a name="create-and-use-a-shared-mailbox"></a>Stvarati i koristiti zajednički poštanski sandučić

Zajednički poštanski sandučić omogućuje slanje i primanje e-pošte za alias, poput **sales@contoso.com**bez dodjele drugu licencu više osoba.
  
[Stvaranje zajednički poštanski sandučić](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9) objašnjava kako: 
  
- Stvaranje zajedničkog poštanskog sandučića
    
- Dati korisnicima dozvole za korištenje zajednički poštanski sandučić
    
- Pristup zajednički poštanski sandučić u programu Outlook ili Outlook na webu
