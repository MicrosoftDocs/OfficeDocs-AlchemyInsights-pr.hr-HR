# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Nadogradnja na kompletnu verziju Microsoftovih timova

Morate nadograditi od slobodnog do pune verzije Microsoftovih timova? Pročitajte članak [Nadogradnja Microsoftovih timova besplatno na verziju pretplate na microsoft 365](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Morat ćete ispuniti ove preduvjete:

- Vi ste osoba koja se prijavila za aktualnu besplatnu verziju timova.
- Ako imate vlastitu domenu, ona se više ne može povezati s servisom Azure Active Directory.
- Da biste prenijeli podatke pri nadogradnji, morate se pretplatiti tijekom postupka nadogradnje. Ako ste pretplatu kupili izvan postupka nadogradnje, ne možete prenijeti podatke.
