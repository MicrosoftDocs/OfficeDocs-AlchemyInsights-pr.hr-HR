# <a name="upgrade-to-the-full-version-of-microsoft-teams"></a>Nadogradnja na punu verziju servisa Microsoft Teams

Trebate nadograditi s besplatnog na punu verziju programa Microsoft Teams? Pogledajte [Besplatna nadogradnja servisa Microsoft Teams na verziju pretplate na Microsoft 365](https://docs.microsoft.com/microsoftteams/upgrade-freemium)

Morat ćete ispuniti ove preduvjete:

- Vi ste osoba koja se prijavila za trenutnu besplatnu verziju programa Teams.
- Ako imate vlastitu domenu, ona se već ne može povezati sa servisom Azure Active Directory.
- Da biste prenijeli podatke prilikom nadogradnje, morate se pretplatiti tijekom postupka nadogradnje. Ako ste pretplatu kupili izvan postupka nadogradnje, ne možete prenositi podatke.
