# <a name="activating-windows-10-subscriptions"></a>Aktiviranje pretplata na Windows 10

Microsoft 365 Business Premium, Microsoft 365 Education, Microsoft 365 i Enterprise u sklopu pretplate pružaju digitalne licence za Windows 10. Za aktiviranje pretplate vaš uređaj mora već imati aktiviran Windows 10 Pro (1703 ili noviju verziju). Potpuni popis preduvjeta za aktivaciju pretplate na Windows 10 potražite [u odjeljku Preduvjeti za aktivaciju pretplate u sustavu Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
