# <a name="activating-windows-10-subscriptions"></a>Aktivacija pretplata na Windows 10

Microsoft 365 Business, Microsoft 365 obrazovanje, Microsoft 365 i Enterprise pružaju digitalne licence za Windows 10 kao dio pretplate. Da biste aktivirali pretplatu, vaš uređaj mora već biti aktiviran Windows 10 Pro (1703 ili noviji). Da biste vidjeli potpuni popis zahtjeva za aktivaciju pretplate na Windows 10, pogledajte [preduvjete za aktivaciju pretplate na Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
