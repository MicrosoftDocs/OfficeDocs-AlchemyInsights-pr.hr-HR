# <a name="how-microsoft-edge-uses-microsoft-defender-smartscreen"></a>Kako Microsoft Edge koristi Microsoft Defender SmartScreen

Da bi vam bio siguran na webu, Microsoft Edge koristi Microsoft Defender SmartScreen kao sustav za rano upozoravanje na web-mjesta koja poиinju napade krađe identiteta i distribuiraju zlonamjerni softver kroz fokusirane napade.

Administratori mogu konfigurirati Microsoft Defender SmartScreen pomoću pravilnika grupe, Microsoftova Intune ili servisa za upravljanje mobilnim uređajima (MDM). Na temelju načina na koji postavljate Microsoft Defender SmartScreen, korisnicima možete prikazati stranicu s upozorenjima i dopustiti im da nastave s web-mjestom ili u cijelosti blokiraju web-mjesto.

Dodatne informacije potražite u [članku Podrška za Microsoft Edge za Microsoft Defender SmartScreen](https://go.microsoft.com/fwlink/?linkid=2133081) i [Microsoft Defender Guard](https://go.microsoft.com/fwlink/?linkid=2132839).
