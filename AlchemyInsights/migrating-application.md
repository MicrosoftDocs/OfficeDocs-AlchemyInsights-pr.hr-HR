# <a name="migrating-applications"></a>Migriranje aplikacija

[Resursi za migriranje aplikacija u Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migration-resources) – resursi koji će vam pomoći u migraciji pristupa aplikacijama i provjere autentičnosti na Azure Active Directory (Azure AD).

[Premještanje provjere autentičnosti aplikacija iz servisa Active Directory Federation u Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/migrate-adfs-apps-to-azure) – developer usmjereno smjernice za migraciju aplikacija iz servisa Active Directory Federation Services u Azure Active Directory.
