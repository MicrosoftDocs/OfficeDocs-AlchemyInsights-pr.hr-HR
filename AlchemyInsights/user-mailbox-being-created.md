# <a name="your-users-mailbox-is-being-created"></a>Stvara se poštanski sandučić vašeg korisnika

Može potrajati i do sat vremena prije nego što novi poštanski sandučić bude spreman za korištenje.
  
Ako je korisnikov poštanski sandučić "priprema" više od jednog sata, možete pokušati izbrisati i ponovno stvoriti korisnika ili koristiti mogućnost "Nazovimo vas" da biste razgovarali s inženjerom podrške.
