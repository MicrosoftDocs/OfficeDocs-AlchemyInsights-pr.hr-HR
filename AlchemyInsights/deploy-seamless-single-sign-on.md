# <a name="enable-seamless-sso"></a>Omogući neometano SSO

Azure Active Directory (Azure AD) besprijekorna jednokratna prijava (bešavni SSO) automatski se prijavljuje kod korisnika kada su na svojim korporativnim radnim površinama koje su povezane s vašom korporativnom mrežom.
  
[Slijedite korake u ovom članku kako biste omogućili nesmetano SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
