# <a name="enable-seamless-sso"></a>Omogući besprijekorni SSO

Azure Active Directory (Azure AD) besprijekorna jedinstvena prijava (bešavni SSO) automatski se prijavljuje u korisnike kada se nalaze na svojim korporacijskim radnim površinama koje su povezane s vašom korporacijskom mrežom.
  
[Slijedite korake u ovom članku da biste omogućili besprijekoran SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
