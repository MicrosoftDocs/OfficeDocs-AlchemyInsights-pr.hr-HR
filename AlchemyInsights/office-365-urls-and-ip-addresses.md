# <a name="microsoft-365-urls-and-ip-address-ranges"></a>URL-ovi Microsoftova 365 i raspona IP adresa

Microsoft 365 potrebna je veza s Internetom putem specifičnih IP adresa koje se nazivaju *krajnje točke*.
Popis krajnjih točaka koje moraju biti dostupne, pročitajte članak [URL-ovi i rasponi IP adresa programa Microsoft 365](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

IP adresa Microsoftova 365 i web-servis URL-a pomoći će vam da bolje prepoznajete i diferentišete mrežni promet u sustavu Microsoft 365, što vam olakšava vrednovanje, konfiguriranje i održavanje ažurnih promjena. Pročitajte članak [IP adresa i URL web-servisa Microsoft 365](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
