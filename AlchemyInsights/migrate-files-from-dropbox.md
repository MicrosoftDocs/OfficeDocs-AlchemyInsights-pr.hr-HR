# <a name="migrate-files-from-dropbox"></a>Migriranje datoteka iz programa Dropbox

Ako migrirate iz servisa Dropbox u SharePoint Online, možete razmotriti korištenje servisa **pokretač**.

Dodatne informacije potražite u članku [autorizacija servisa Dropbox](https://docs.microsoft.com/sharepointmigration/mover-dropbox).
