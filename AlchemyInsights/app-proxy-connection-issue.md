# <a name="app-proxy-connection-issue"></a>Problem s proxy vezom aplikacija

1. Ako koristite poslužitelj za proxy aplikacije za udaljeni pristup lokalnoj web-aplikaciji, ali imate problema s povezivanjem s aplikacijom, pomoću [ovog članka](https://docs.microsoft.com/azure/active-directory/manage-apps/application-proxy-debug-connectors) možete otkloniti poteškoće s poveznicima proxy poslužitelja aplikacije Azure Active Directory (Azure AD).
2. Pomoć za rješavanje običnih problema s kojima se suočavaju kada se poveznik ne prepozna za proxy aplikaciju aplikacije integriranu sa servisom Azure Active Directory, slijedite korake u članku nema radne veze koja se [nalazi u grupi za proxy](https://docs.microsoft.com/azure/active-directory/application-proxy-connectivity-no-working-connector) aplikacije koja će se otkloniti.
