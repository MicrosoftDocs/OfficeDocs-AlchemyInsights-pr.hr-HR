# <a name="desktop-app-for-sharepoint"></a>Aplikacija desktop za SharePoint

Vaša je organizacija unesena u program privatnog pregleda za aplikaciju desktop za SharePoint. Microsoft trenutno ne podržava aplikaciju desktop za SharePoint (Private Preview). Upotrijebite svoje regrutne partnere za Microsoft Private Preview da biste dobili pomoć s aplikacijom za desktop za SharePoint.
