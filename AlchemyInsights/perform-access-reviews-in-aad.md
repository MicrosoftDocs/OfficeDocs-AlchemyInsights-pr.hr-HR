# <a name="perform-access-reviews-in-azure-active-directory"></a>Izvršavanje recenzija programa Access u servisu Azure Active Directory

Označeni pregledavatelji mogu izvršavati recenzije servisa Azure Active Directory i pristupati članovima grupe, pristupu aplikacijama i dodjelama uloga. Ove recenzije omogućuju smanjivanje sigurnosnih rizika s kojima se vaša tvrtka ili ustanova može suočiti kada korisnici imaju više pristupa nego što im je potrebno. Možete obavljati recenzije programa Access pomoću [mojih aplikacija](https://go.microsoft.com/fwlink/?linkid=2134605) ili [mog pristupa](https://go.microsoft.com/fwlink/?linkid=2134505).
