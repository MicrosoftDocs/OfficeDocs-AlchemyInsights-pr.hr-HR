# <a name="solutions-for-unsupported-operating-system"></a>Rješenja za "Nepodržan operacijski sustav"

Kada pokušate instalirati Office 365, ako se vaš uređaj ne udovoljava [zahtjevima sustava](https://products.office.com/office-system-requirements)možda vidjeti poruku o pogrešci **operacijski sustav nije podržan** .
  
Office 365 zahtijeva:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 ili Windows Server 2008 R2

    ILI

- Mac OS X 10.10

Dobio ovu poruku tijekom pokušaja instalirajte Office na mobilni uređaj ili u Chromebook, pogledajte [kako instalirati i pokrenuti Microsoft Office na u Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
