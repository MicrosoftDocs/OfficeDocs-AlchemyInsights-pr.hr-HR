# <a name="solutions-for-unsupported-operating-system"></a>Rješenja za "Nepodržan operacijski sustav"

Kada pokušate instalirati Office 365 2016 možda vidjeti sljedeću poruku o pogrešci ako uređaj ne udovoljava [zahtjevima sustava](https://products.office.com/office-system-requirements).
  
 **"Nepodržan operacijski sustav"**
  
Office 365 2016 zahtijeva:
  
PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 ili Windows Server 2008 R2
  
Umjesto toga,
  
Mac OS X 10.10
  
Dobio ovu poruku tijekom pokušaja instalirajte Office na mobilni uređaj ili u Chromebook, pogledajte [Postavljanje Office apps i e-poštu na mobilni uređaj](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
