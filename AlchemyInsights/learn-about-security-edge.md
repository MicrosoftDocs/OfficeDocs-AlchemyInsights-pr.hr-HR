# <a name="learn-about-the-security-features-of-microsoft-edge"></a>Saznajte više o sigurnosnim značajkama preglednika Microsoft Edge

It administratori za velike tvrtke bave se mnoštvom postojećih i novih izazova povezanih sa sigurnošću. Rješavanje tih izazova administratori moraju učiniti da bi korporacijsku mrežu i uređaje zaštitili od zlonamjernih napada te spriječili neovlašteni pristup korporativnim podacima i curenje podataka. Da biste administratorima pomogli u rješavanju tih izazova, Microsoft Edge (verzija 77 ili novija) nudi nekoliko jedinstvenih ugrađenih mogućnosti. 

Dodatne informacije potražite u članku [Sigurnost preglednika Microsoft Edge za svoju tvrtku](/DeployEdge/ms-edge-security-for-business).
