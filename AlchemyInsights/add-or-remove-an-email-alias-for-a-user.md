# <a name="add-or-remove-an-email-address-for-a-user"></a>Dodavanje ili uklanjanje adrese e-pošte za korisnika

Da biste [započeli, slijedite ove korake](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) u centru za administratore sustava Microsoft 365.

 **Napomena:** možete imati više adresa e-pošte za korisnika, ali dodatni *pseudonimi e-pošte* mogu se koristiti samo za primanje e-pošte. Ako trebate poslati e-poštu s pseudonimom, stvorite [zajednički poštanski sandučić](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox). Na taj ćete način dobiti pseudonim za slanje i primanje pošte.
