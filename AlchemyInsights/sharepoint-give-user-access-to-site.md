# <a name="give-users-access-to-sharepoint-site"></a>Daj korisnicima pristup SharePoint web-mjesta

Značajka zahtjev access omogućuje korisnicima zatražili pristup sadržaju trenutno nemaju dozvolu za prikaz. Kao vlasnik web-mjesta možete konfigurirati značajku slanje pošte vas kada netko zahtijeva pristup web-mjestu. Za dodatne informacije posjetite [postavite gore i Upravljanje zahtjevima za pristup](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).
