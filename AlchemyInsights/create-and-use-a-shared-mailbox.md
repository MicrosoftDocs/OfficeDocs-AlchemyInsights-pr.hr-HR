# <a name="create-and-use-a-shared-mailbox"></a>Stvarati i koristiti zajednički poštanski sandučić

Zajednički poštanski sandučić omogućuje slanje i primanje e-pošte za alias, poput sales@contoso.com, bez dodjele drugu licencu više osoba.
  
- [Idite tamo da biste započeli](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions)

Možete pročitati u članku [Stvaranje zajednički poštanski sandučić](https://support.office.com/article/Create-a-shared-mailbox-871a246d-3acd-4bba-948e-5de8be0544c9.aspx) koji će vam reći sve što trebate znati o zajednički poštanski sandučići, uključujući kako ih koristiti.
