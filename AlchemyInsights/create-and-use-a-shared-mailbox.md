# <a name="create-and-use-a-shared-mailbox"></a>Stvaranje i korištenje zajedničkog poštanskog sandučića

Zajednički poštanski sandučić omogućuje više od jedne osobe slanje i primanje e-pošte za pseudonim, kao što je sales@contoso.com, bez dodjele druge licence.
  
- Da biste započeli, [slijedite ove korake](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) u centru za administratore sustava Microsoft 365. 

Možete pročitati i članak [Stvaranje zajedničkog poštanskog sandučića](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox) koji će vam reći sve što trebate znati o zajedničkim poštanskim sandučićima, uključujući i kako ih koristiti.
