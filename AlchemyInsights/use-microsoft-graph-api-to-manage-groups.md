# <a name="use-microsoft-graph-api-to-manage-groups"></a>Upravljanje grupama pomoću API-ja programa Microsoft Graph

Grupe su zbirke korisnika i drugih nadređenima koji dijele pristup resursima u Microsoftovim servisima ili u aplikaciji. Microsoft Graph sadrži API-je koje možete koristiti za stvaranje različitih vrsta grupa i funkcionalnosti grupe prema scenariju. Sve operacije vezane uz grupu u programu Microsoft Graph zahtijevaju pristanak administratora. Dodatne informacije potražite u članku [rad s grupama u programu Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview).
