# <a name="live-events-in-microsoft-stream"></a>Događaji uživo u programu Microsoft Stream

Događaje uživo možete stvarati pomoću Microsoft strujanja u cijeloj tvrtki ili ustanovi. Možete zakazati, proizvoditi i davati događaje uživo za razne scenarije, kao što su događaji u širokom društvu, ažuriranja vodstva i još mnogo toga.

- [Stvaranje događaja uživo u programu Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Početak rada s događajima u programu Microsoft klubovi Live](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Konfigurirajte dekoderi za Live streaming u programu Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
