# <a name="windows-defender-actions-in-intune"></a>Akcije programa Windows Defender u sustavu Intune

Intune se može koristiti za pokretanje ažuriranja skeniranja na zahtjev i virusnog potpisa za Windows Defender na pojedinačnim uređajima.

Nakon uspješnog pokretanja udaljene akcije aktivnost se odražava u zapisniku događaja programa Windows Defender.

Pravila zaštite krajnjih točaka sustava Windows omogućuju stvaranje dodatnih postavki za stvaranje značajki programa Windows Defender u aplikaciji Intune i primjenu na skupove uređaja.

Dodatne informacije o pokretanju akcija programa Windows Defender [potražite u odjeljku Konfiguriranje i pokretanje skeniranja programa Microsoft Defender Antivirus na zahtjev](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/run-scan-windows-defender-antivirus).
