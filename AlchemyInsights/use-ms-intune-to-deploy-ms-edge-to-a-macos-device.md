# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Korištenje aplikacije Microsoft Intune za implementaciju sustava Microsoft Edge na macOS uređaj

Da biste uveli, konfigurirali, nadzirali i zaštitili aplikaciju na macOS uređaju, morate dodati aplikaciju u Microsoft Intune. Aplikacija mora biti vrsta koju podržava Intune, a mogućnosti upravljanja dostupne u programu Intune ovisit će o vrsti aplikacije. Ako koristite Intune, to znači da nećete morati koristiti alat za prelamanje aplikacija macOS.

Intune podržava Microsoft Edge. Dodatne informacije potražite u članku [Dodavanje programa Microsoft Edge na macOS uređaje pomoću aplikacije Microsoft Intune](https://go.microsoft.com/fwlink/?linkid=2134949).
