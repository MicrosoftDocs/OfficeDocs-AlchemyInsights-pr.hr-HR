# <a name="what-is-autosave"></a>Što je automatsko spremanje?

Automatsko spremanje značajka je dostupna u pretplatnicima programa Excel, Word i PowerPoint za Pretplatnike na Microsoft 365 koji automatski sprema datoteku svakih nekoliko sekundi dok radite. 

Najčešća pitanja potražite u odjeljku [Što je automatsko spremanje?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
