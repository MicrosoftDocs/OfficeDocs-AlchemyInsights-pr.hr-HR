# <a name="sharepoint-online-throttling"></a>Ograničavanje servisa SharePoint Online

SharePoint Online koristi ograničavanje za održavanje optimalnih performansi i pouzdanosti servisa SharePoint Online. Ograničavanjem se ograničava broj korisničkih radnji ili istodobnih poziva (skriptom ili kodom) radi sprječavanja prekomjernog korištenja resursa. Ako se ugušite, 99% vremena je zbog prilagođenog koda.
  
Dodatne informacije o regulaciji [potražite u članku Izbjegavanje ograničavanja ili blokiranja u sustavu SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
