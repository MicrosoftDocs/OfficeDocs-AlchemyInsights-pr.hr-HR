# <a name="sharepoint-online-throttling"></a>Online Reguliranje SharePoint

SharePoint Online koristi Reguliranje za održavanje optimalnih performansi i pouzdanosti usluge SharePoint Online. Reguliranje ograničava broj akcije korisnika ili Istodobni poziva (po skripta ili kod) spriječiti Prekomjerno korištenje resursa. Ako ste dobili ograničio vrijeme, 99% vrijeme je radi prilagođenog koda.
  
Dodatne informacije o Reguliranje pogledajte [Avoid dobivanju ograničio vrijeme ili blokiran u SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
