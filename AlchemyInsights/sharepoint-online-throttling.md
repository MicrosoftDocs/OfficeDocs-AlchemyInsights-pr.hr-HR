# <a name="sharepoint-online-throttling"></a>Ograničavanje sustava SharePoint Online

SharePoint Online koristi ograničavanje za održavanje optimalne performanse i pouzdanost usluge SharePoint online. Reguliranje ograničava broj korisničkih akcija ili istodobnih poziva (po skripti ili kodu) kako bi se spriječilo prekomjerno korištenje resursa. Ako se ograniči, 99% vremena je zbog prilagođenog koda.
  
Dodatne informacije o regulaciji Pogledajte, [Izbjegavajte ograničavanje ili blokiranje u SharePoint Online](https://go.microsoft.com/fwlink/?linkid=2022019)
