# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Pogreška programa Outlook: "Nije moguće pokrenuti Microsoft Outlook"

Imate li problema s pokretanjem programa Outlook 2016, Outlook 2013 ili Outlook 2010 ili primanjem "Nije moguće pokrenuti Microsoft Outlook. Nije moguće otvoriti poruke o pogrešci outlook prozora"?
  
Najprije slijedite korake za otklanjanje poteškoća u [ovom članku](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) da biste riješili te probleme pri pokretanju. 
  
Alat [za naprednu dijagnostiku](https://aka.ms/SaRA-OutlookAdvDiagnostics) možete pokrenuti i na zahvaćenom računalu.
