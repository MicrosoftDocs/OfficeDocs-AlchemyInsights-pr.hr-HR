# <a name="microsoft-stream-licensing-overview"></a>Pregled licenciranja za Microsoft Stream

Microsoft Stream je usluga u oblaku licencirana po korisnikovoj pretplati. Dostupan je kao dio pretplata na Office 365 koji ispunjavaju uvjete. Informacije o svakoj vrsti pretplate potražite u članku Određivanje cijena sustava Office 365.

- [Pregled licenciranja za Microsoft Stream](https://docs.microsoft.com/stream/license-overview)
